# 🎓 CPI/SPI Calculator Dashboard

A modern and interactive **Streamlit web application** that helps students calculate the **Semester Performance Index (SPI)** required to achieve their desired **Cumulative Performance Index (CPI)**.

Designed with a clean UI and step-by-step workflow, this dashboard makes academic planning simple and intuitive.

---

## 🌐 Live Application

**Try it here:**

## ✨ Key Features

- 🎯 Calculate the SPI required to achieve your target CPI.
- 📚 Supports multiple remaining semesters.
- 📈 Instant calculations with real-time updates.
- 🎨 Modern and responsive user interface.
- ⚡ Built using Python and Streamlit.
- 💻 Works directly in the browser without installation.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Prabhatpetro/CPI-Calculator-Dashboard.git
```

Move into the project directory

```bash
cd CPI-Calculator-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
